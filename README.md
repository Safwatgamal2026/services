# Turki General Services

## English Description

Turki General Services is a bilingual (Arabic/English) landing page designed to present professional public services in a clear and modern style. The page includes:

- A responsive hero section with clear call-to-action buttons.
- A services section with premium cards and interactive visual effects.
- A "Why Us" section highlighting quality, speed, competitive pricing, and team expertise.
- A contact section with phone, WhatsApp, location, and a direct WhatsApp request form.
- Full responsiveness for desktop, tablet, and mobile with language switching support.

## الوصف باللغة العربية

موقع تركي للخدمات العامة هو صفحة تعريفية ثنائية اللغة (العربية/الإنجليزية) مصممة لعرض الخدمات العامة بشكل احترافي وعصري. تتضمن الصفحة:

- قسم رئيسي جذاب مع أزرار واضحة للتواصل واستكشاف الخدمات.
- قسم الخدمات بكاردات مميزة مع تأثيرات بصرية تفاعلية.
- قسم "لماذا نحن" لإبراز الجودة، سرعة الاستجابة، الأسعار التنافسية، وكفاءة الفريق.
- قسم تواصل يحتوي على الهاتف، واتساب، الموقع، ونموذج إرسال مباشر عبر واتساب.
- توافق كامل مع جميع الشاشات (ديسكتوب، تابلت، جوال) مع دعم التبديل بين اللغتين.
